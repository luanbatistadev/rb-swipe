## Regras de Código

1. **Sem comentários**: O código deve ser autoexplicativo. Não usar comentários (nem `//` nem `///`).
2. **Nomes descritivos**: Usar nomes de variáveis e funções que expliquem seu propósito.
3. **Código limpo**: Manter o código simples e direto.
